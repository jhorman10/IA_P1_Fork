# domain
- Usuario (user): persona autenticada via Firebase
- Perfil (profile): datos personales + config
- UID (uid): id de Firebase Auth (nunca generar)
- FAQ: par pregunta-respuesta publicado
- Pregunta (question), Respuesta (answer)
- Dashboard: pantalla principal, solo lectura
- Modo Oscuro (dark mode): tema alternativo
- Token (idToken): Firebase, header Authorization: Bearer
- Administrador: rol con permisos completos
- created_at, updated_at: snake_case, UTC
rejected_synonyms: [persona, cliente, _id, ID técnico, sesión, contraseña, modo noche]
